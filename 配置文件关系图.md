## 🎯 数据库访问流程图

```
用户点击"连接数据库"
       │
       ▼
frontend/js/app.js
发送 POST /api/connect-db
       │
       ▼
backend/routes/api.py
@api.route('/connect-db')
       │
       ▼
backend/models/models.py
create_engine_and_session()
       │
       ▼
backend/config/database.py
读取 DB_CONFIG
       │
       ├─→ db_type = 'postgresql'
       ├─→ schema = 'crawled-zxb-data'
       │
       ▼
构建连接 URL:
postgresql://user:pass@host:port/database
       │
       ▼
设置 search_path:
-c search_path=crawled-zxb-data
       │
       ▼
PostgreSQL 数据库
192.168.200.88:5432
       │
       └─→ 自动在 crawled-zxb-data schema 下
           查找 questions, replies, images 表
```

## 📤 Excel 数据处理流程

```
用户上传 Excel 文件
       │
       ▼
frontend/js/app.js
发送 POST /api/upload-excel
       │
       ▼
backend/routes/api.py
@api.route('/upload-excel')
       │
       ▼
backend/utils/excel_parser.py
parse_excel_file(file_path)
       │
       ▼
backend/config/database.py
读取 EXCEL_COLUMN_MAPPING
       │
       ├─→ questions 列映射
       ├─→ replies 列映射
       └─→ images 列映射
       │
       ▼
验证列名是否匹配
       │
       ├─→ 成功 → 返回数据
       └─→ 失败 → 返回错误
       │
       ▼
backend/services/scoring.py
score_all_replies(replies_list)
       │
       ▼
backend/config/database.py
读取 SCORING_RULES
       │
       ├─→ accepted_flag 权重
       ├─→ user_type 权重
       └─→ content_length 阈值
       │
       ▼
计算每条回复的评分
       │
       ▼
返回评分结果给前端
```

##  数据导出流程

```
用户点击"导出 JSON"
       │
       ▼
frontend/js/app.js
发送 POST /api/export-database
       │
       ▼
backend/routes/api.py
@api.route('/export-database')
       │
       ▼
backend/services/export.py
export_from_database(session)
       │
       ├─→ 查询 questions 表
       ├─→ 查询 replies 表 (evl > 0)
       └─→ 查询 images 表
       │
       ▼
prepare_training_data()
       │
       ├─→ 按 question_no 分组
       ├─→ 为每个问题选择 evl 最高的回复
       └─→ 收集所有关联的图片
       │
       ▼
extract_filename_from_url()
从 URL 提取文件名
       │
       ▼
构建 Qwen 训练格式
{
  "id": "question_no",
  "image": ["file1.jpeg", "file2.jpeg"],
  "conversations": [...]
}
       │
       ▼
backend/config/database.py
OUTPUT_FOLDER = 'outputs'
       │
       ▼
保存到 outputs/
training_data_YYYYMMDD_HHMMSS.json
       │
       ▼
返回下载链接给前端
```

## 🔑 关键配置项总结

| 配置项 | 位置 | 被谁使用 | 作用 |
|--------|------|---------|------|
| `DB_CONFIG` | `config/database.py` | `models.py` | 数据库连接参数 |
| `DB_CONFIG['schema']` | `config/database.py` | `models.py` | 指定 PostgreSQL Schema |
| `DB_CONFIG['db_type']` | `config/database.py` | `models.py` | 选择数据库类型 |
| `EXCEL_COLUMN_MAPPING` | `config/database.py` | `excel_parser.py` | Excel 列名映射 |
| `SCORING_RULES` | `config/database.py` | `scoring.py` | 评分规则定义 |
| `UPLOAD_FOLDER` | `config/database.py` | `excel_parser.py` | 上传文件目录 |
| `OUTPUT_FOLDER` | `config/database.py` | `export.py` | 导出文件目录 |
